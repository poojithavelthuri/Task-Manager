# Task Manager App

## Tech Stack

- Java 17
- Spring Boot
- Spring Data JPA
- MySQL
- HTML, CSS, JavaScript
- Bootstrap

## Getting Started

- Create MySQL DB:
```sql
CREATE DATABASE task_db;
```

- Update credentials in `application.properties`
- Run the backend using:
```bash
./mvnw spring-boot:run
```
- Open `index.html` in your browser.
